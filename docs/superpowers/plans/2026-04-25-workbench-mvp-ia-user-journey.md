## 2. 历史摘要

本文件曾承载 W13-B / W13-FC 后的一期工作台 MVP IA 与用户旅程结论，确认主链为：

`模拟记录列表 -> 发起模拟面试 -> 面试台 -> 评分 / 复盘详情 -> 回到模拟记录列表`

已迁入正式事实源的核心 IA 包括：

- 工作台、模拟面试、岗位、简历、知识库、复盘 / 评分、账号 / 权限等一级入口。
- 从模拟记录、岗位详情、复盘训练、真实面试材料和知识库资料进入工作台闭环。
- RAG / 知识库与多轮高阶面试进入一期主链，不再作为后续默认占位。

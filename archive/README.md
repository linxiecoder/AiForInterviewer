# archive

`archive/` 只保存历史来源、废弃文档、审计证据和归档台账。archive 内容不得作为当前需求、设计、阶段、任务或实现依据。

## 使用规则

- 归档动作必须登记到 `archive/MANIFEST.md`。
- 历史内容如仍有效，必须先迁入 active docs。
- 历史需求继承只通过 `docs/01-product/REQUIREMENT_TRACEABILITY.md` 追踪。
- 归档审计报告只作证据，不替代当前治理文档。
